## Quick Summary

**What is this product?**  
A markdown-native, multi-interface solution for weekly and daily planning, task management, and habit tracking. The system is built on Git-backed markdown files, supports direct editing in any text editor, and provides web/mobile PWA and CLI interfaces.

**MVP Release Includes:**
- Weekly and daily planning with time blocking
- Task inbox and task management (with projects, areas, contexts)
- Basic habit tracking (frequency-based with completion tracking)
- Recurring tasks and reminders
- Multiple interfaces: markdown, web app (PWA), CLI

**Future Enhancements:**
- Values/life goals and multi-tier planning (annual, quarterly)
- End-of-day reflection (Marshall Goldsmith's "Triggers" method)
- Calendar integration
- Inventory tracking
- AI agent skills and integration
- Advanced analytics and insights
